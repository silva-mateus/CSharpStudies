# Methods - AbsoluteOfSum

## Description
Define the `AbsoluteOfSum` method that takes two `int` parameters and returns the absolute value of their sum.

## Explanation
Absolute value is the non-negative value of a number without regard to its sign.
- `|5| = 5`
- `|-10| = 10`

## Topics Covered
- Method definition
- Parameters and return types
- Absolute value (`Math.Abs()`)
