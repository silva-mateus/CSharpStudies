# String Interpolation - FormatDate

## Description
Using string interpolation, implement the `FormatDate` method that formats three integers (year, month, day) into a string in the format `YEAR/MONTH/DAY`.

## Examples
- `FormatDate(2022, 10, 29)` → `"2022/10/29"`
- `FormatDate(1911, 1, 5)` → `"1911/1/5"`

## Topics Covered
- String interpolation
- Method parameters
- String formatting
