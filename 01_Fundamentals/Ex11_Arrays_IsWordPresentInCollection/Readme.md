# Arrays (2) - IsWordPresentInCollection

## Description
Implement the `IsWordPresentInCollection` method that checks if a word exists in an array of strings.

## Requirements
Returns `true` if `wordToBeChecked` is present in the `words` array, `false` otherwise.

## Examples
- `words = {"one", "two", "three"}`, `wordToBeChecked = "hello"` → `false`
- `words = {"one", "two", "three"}`, `wordToBeChecked = "two"` → `true`
