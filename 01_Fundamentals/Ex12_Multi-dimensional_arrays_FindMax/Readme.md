# Multi-dimensional Arrays - FindMax

## Description
Implement the `FindMax` method that finds the maximum value in a two-dimensional array of integers.

## Requirements
- If any dimension is zero (empty array), return `-1`
- Otherwise, return the maximum value found

## Examples
- Empty array `[0,3]` → `-1`
- Array with values → maximum value (e.g., `12`)

## Topics Covered
- Multi-dimensional arrays
- Nested loops
- Array boundary checking
- Finding maximum values
