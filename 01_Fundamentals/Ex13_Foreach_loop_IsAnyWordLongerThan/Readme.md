# Foreach Loop - IsAnyWordLongerThan

## Description
Using a `foreach` loop, implement the `IsAnyWordLongerThan` method that checks if any word in an array exceeds a given length.

## Requirements
Returns `true` if any word is longer than the specified length, `false` otherwise.

## Examples
- `length = 4`, `words = {"aaa", "bbbb", "ccccc"}` → `true` ("ccccc" has 5 letters)
- `length = 5`, `words = {"aaa", "bbbb", "ccccc"}` → `false` (no word exceeds 5 letters)
- `length = 5`, `words = {}` (empty array) → `false`

## Topics Covered
- Foreach loops
- String length property
- Boolean logic
- Early return pattern
