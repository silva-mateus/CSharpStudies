# Static Classes - NumberToDayOfWeekTranslator

## Description
Implement a static `NumberToDayOfWeekTranslator` class with a static `Translate` method.

## Requirements

### Method Signature
```csharp
public static string Translate(int dayNumber)
```

### Number-to-Day Mapping
- `1` → `"Monday"`
- `2` → `"Tuesday"`
- `3` → `"Wednesday"`
- `4` → `"Thursday"`
- `5` → `"Friday"`
- `6` → `"Saturday"`
- `7` → `"Sunday"`
- Any other number → `"Invalid day of the week"`
